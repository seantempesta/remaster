# Remaster DRUNet Teacher -- High-Quality Video Enhancement

A 32.6M parameter DRUNet model that removes compression artifacts and recovers detail from 1080p video. Produces output that exceeds the quality of the original Bluray source material (107% sharpness). Used as the teacher for distilling the real-time student model.

## Model Details

| Property | Value |
|----------|-------|
| **Architecture** | DRUNet (UNetRes from KAIR) |
| **Parameters** | 32.6M |
| **Channels** | nc=[64, 128, 256, 512] |
| **Residual blocks per level** | nb=4 |
| **Activation** | ReLU (act_mode='R') |
| **Bias** | False |
| **Input** | 3-channel RGB [0, 1], spatial dims divisible by 8 |
| **Output** | 3-channel RGB [0, 1] |
| **Checkpoint size** | 125 MB |
| **License** | MIT |

### Architecture Description

4-level encoder-decoder U-Net with skip connections. Pure Conv 3x3 + ReLU residual blocks at each level. No batch normalization, no layer normalization, no attention. Channel progression: 64 -> 128 -> 256 -> 512. Pretrained from `drunet_deblocking_color.pth` (Gaussian denoising weights).

## Performance

| Metric | Value |
|--------|-------|
| **PSNR** | 53.27 dB (vs SCUNet GAN + USM targets) |
| **Sharpness** | 107% of original source |
| **Speed (FP16, RTX 3060)** | ~5 fps |
| **VRAM** | ~2 GB |
| **Episode (44 min, 1080p)** | ~2.5 hours |

## Training

### Targets

Training targets generated by [SCUNet GAN](https://github.com/cszn/SCUNet) with Unsharp Mask (strength=1.0). Color-corrected: GAN luminance (Y channel) combined with original chrominance (CrCb) to avoid color shifts.

### Loss Functions

- **Charbonnier** -- smooth L1 pixel loss
- **[DISTS](https://github.com/dingkeyan93/DISTS)** (weight 0.15) -- perceptual loss

### Optimizer

Prodigy (auto-tuned learning rate) with EMA weight averaging. 13K+ iterations.

### Data

~7K paired frames from diverse 1080p content (6 sources: Firefly, The Expanse, One Piece, Squid Game, Dune Part Two, Foundation). Input degradation mix: 33% raw HEVC originals, 33% +noise, 33% +edge-aware blur+noise.

## Primary Use Cases

1. **Distillation teacher** -- generates soft targets for training the real-time student model via knowledge distillation with feature matching
2. **Quality reference** -- establishes the quality ceiling for student training and evaluation
3. **Offline enhancement** -- when speed is not critical and maximum quality is desired

## Usage

```python
import torch
import sys
sys.path.insert(0, "/path/to/KAIR")  # github.com/cszn/KAIR
from models.network_unet import UNetRes

model = UNetRes(in_nc=3, out_nc=3, nc=[64, 128, 256, 512], nb=4,
                act_mode='R', bias=False)
ckpt = torch.load("drunet_teacher.pth", map_location="cpu", weights_only=True)
model.load_state_dict(ckpt["params"])
model.eval().half().cuda()

with torch.no_grad(), torch.cuda.amp.autocast():
    output = model(input_tensor)
output = output.clamp(0, 1)
```

## Key Finding

Mixed training data (HEVC artifact removal + synthetic edge-aware blur) produces a model that generalizes beyond its training tasks. The teacher denoises AND sharpens, reaching 107% of the original source material's sharpness -- exceeding Bluray quality.
