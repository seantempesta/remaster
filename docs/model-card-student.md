# Remaster DRUNet Student -- Real-Time Video Enhancement

A tiny 1.06M parameter neural network that removes compression artifacts AND recovers detail from 1080p video at 57 fps on an RTX 3060. Distilled from a 32.6M parameter teacher model via knowledge distillation with feature matching.

## Model Details

| Property | Value |
|----------|-------|
| **Architecture** | DRUNet (UNetRes from KAIR) |
| **Parameters** | 1.06M |
| **Channels** | nc=[16, 32, 64, 128] |
| **Residual blocks per level** | nb=2 |
| **Activation** | ReLU (act_mode='R') |
| **Bias** | False |
| **Input** | 3-channel RGB [0, 1], spatial dims divisible by 8 |
| **Output** | 3-channel RGB [0, 1] |
| **Checkpoint size** | 4 MB |
| **License** | MIT |

### Architecture Description

4-level encoder-decoder U-Net with skip connections. Pure Conv 3x3 + ReLU residual blocks at each level. No batch normalization, no layer normalization, no attention -- entirely static Conv+ReLU operations. This makes the model 100% compatible with TensorRT INT8 quantization and CUDA graph capture.

Channel progression through encoder levels: 16 -> 32 -> 64 -> 128. Downsampling via strided convolution, upsampling via transposed convolution. Global residual connection (model learns the residual correction).

## Training

### Distillation

The student was distilled from a 32.6M parameter DRUNet teacher (nc=[64,128,256,512], nb=4) that achieves 53.27 dB PSNR. The teacher was trained on ~7K paired frames from diverse 1080p content (live action, animation, film).

### Targets

Training targets were generated by [SCUNet GAN](https://github.com/cszn/SCUNet) (a perceptual/adversarial denoiser) with Unsharp Mask (strength=1.0) applied to recover sharpness. This combined denoise + sharpen target teaches the model to go beyond artifact removal into detail recovery.

### Loss Functions

- **Charbonnier** -- smooth L1 pixel loss for overall fidelity
- **[DISTS](https://github.com/dingkeyan93/DISTS)** (weight 0.05) -- perceptual loss calibrated to human perception of compression artifacts
- **Feature Matching** (weight 0.1) -- L1 distance between student and teacher encoder features at each U-Net level, using learned 1x1 adapter convolutions

### Fine-Tuning

Initial training used 256x256 random crops. A final fine-tuning pass used full 1920x1080 frames so the model learns how dark edges, letterboxing, and brightness transitions behave across the full frame.

### Optimizer

Prodigy (auto-tuned learning rate, safeguard_warmup, bias_correction) with EMA weight averaging.

### Data

~7K paired frames from diverse 1080p sources:
- Firefly (2002, Fox) -- ~1,876 frames
- The Expanse S02 -- ~1,635 frames
- One Piece S01 (Netflix) -- ~1,306 frames
- Squid Game S02 -- ~1,237 frames
- Dune Part Two -- ~476 frames
- Foundation S03 -- ~426 frames

Input degradation mix: 33% raw HEVC originals, 33% +noise, 33% +edge-aware blur+noise.

## Usage

### Python Inference

```python
import torch
import sys
sys.path.insert(0, "/path/to/KAIR")  # github.com/cszn/KAIR
from models.network_unet import UNetRes

model = UNetRes(in_nc=3, out_nc=3, nc=[16, 32, 64, 128], nb=2,
                act_mode='R', bias=False)
ckpt = torch.load("drunet_student.pth", map_location="cpu", weights_only=True)
model.load_state_dict(ckpt["params"])
model.eval().half().cuda()

# input_tensor: [B, 3, H, W] float16, values in [0, 1], H and W divisible by 8
with torch.no_grad(), torch.cuda.amp.autocast():
    output = model(input_tensor)
output = output.clamp(0, 1)
```

### ONNX Export

python tools/export_onnx.py

### TensorRT Engine Build

```bash
# FP16 engine (one-time, ~2 min)
trtexec --onnx=drunet_student.onnx \
    --shapes=input:1x3x1080x1920 --fp16 --useCudaGraph \
    --saveEngine=drunet_student_1080p_fp16.engine

# INT8 engine (requires calibration data)
trtexec --onnx=drunet_student.onnx \
    --shapes=input:1x3x1080x1920 --int8 --fp16 --useCudaGraph \
    --calib=calibration_data.bin \
    --saveEngine=drunet_student_1080p_int8.engine
```

### NVEncC Encoding (Recommended)

```bash
python remaster/encode_nvencc.py input.mkv output.mkv
```

### VapourSynth Real-Time Playback

```python
import vapoursynth as vs
core = vs.core
clip = core.bs.VideoSource("input.mkv")
clip = core.resize.Bicubic(clip, format=vs.RGBS, matrix_in_s="709")
clip = core.ort.Model(clip, network_path="drunet_student.onnx",
                      backend=core.ort.Backend.TRT(fp16=True))
clip = core.resize.Bicubic(clip, format=vs.YUV420P10, matrix_s="709")
clip.set_output()
```

## Requirements

- PyTorch 2.0+
- [KAIR](https://github.com/cszn/KAIR) for `UNetRes` architecture definition (MIT license)
- NVIDIA GPU with 1GB+ VRAM
- Optional: TensorRT 10+ for optimized inference
- Optional: VapourSynth + [vs-mlrt](https://github.com/AmusementClub/vs-mlrt) for video pipeline

## Limitations

- Single-frame processing (no temporal consistency enforcement)
- Spatial dimensions must be divisible by 8
- Trained primarily on 1080p HEVC content; may not generalize to other codecs or resolutions without fine-tuning
- Detail recovery is bounded by the teacher model's quality ceiling
